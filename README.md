## Publicar en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos y carpetas de este proyecto manteniendo la misma estructura.
3. Ir a Settings > Pages.
4. En Source elegir la rama main y la carpeta / (root).
5. Esperar unos minutos, GitHub da una URL tipo:
   https://Markovicsy.github.io/nombre-repo/
6. Esa URL es la que hay que convertir en código QR para entregar, junto con el link del repositorio.
